# Portfolio Website

Modern portfolio website built with Next.js App Router and Tailwind CSS.

## Development

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Notes

- Uses route groups with `(site)` for public pages.
- API route for contact submissions is available at `/api/contact`.
- Dark mode toggle and smooth scrolling are included.
